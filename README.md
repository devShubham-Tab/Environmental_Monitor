# Environmental_Monitor
Live Link: https://environmental-monitor.onrender.com

A rule-based environmental monitoring system built with Flask that tracks parameters like temperature, humidity, AQI, and pH, and triggers alerts when values exceed defined safety thresholds.

## Features
- Real-time environmental monitoring
- Rule-based alert system
- Dynamic data visualization with charts
- Multi-parameter tracking (Temperature, Humidity, AQI, pH)

## How It Works
- The system continuously generates environmental data
- Values are evaluated using predefined safety rules
- Alerts are triggered when thresholds are exceeded
- Data is visualized in real-time through charts

## Tech Stack
- Backend: Flask (Python)
- Frontend: HTML, CSS, JavaScript
- Charts: Chart.js
- Deployment: Render
